# Todo App

A clean, minimal Todo application built with **Node.js**, **Express**, **MongoDB (Atlas)**, **Passport.js** authentication, **EJS** templating and a modern dark mode UI.

## Features

- User registration & login (username + password)
- Create, complete & delete personal tasks
- Filter tasks: Pending / Completed / All
- Persistent storage in MongoDB
- Responsive design with clean dark theme
- Secure session-based authentication

## Tech Stack

- Backend: Node.js + Express
- Database: MongoDB (Atlas)
- Authentication: Passport.js + passport-local
- Templating: EJS
- Styling: Custom CSS
